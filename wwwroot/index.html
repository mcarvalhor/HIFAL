Welcome to HIFAL!

HIFAL is running now. Please, upload a website or resources in the web root folder. By default, this is the "wwwroot" folder.

Please, be aware of the following remarks:

  • HIFAL doesn't support HTTPS (HTTP over TLS/SSL) yet.
  • HIFAL doesn't serve dynamic content. HIFAL only serves static files. By that, a *.php or *.py file will never be interpreted or executed, but instead downloaded by user's browser.
  • There's no hidden files or folders in HIFAL. Everything under web root folder is visible and available to the web.
  • Folder listing (or directory listing) is enabled in HIFAL. When a user accesses a folder path, first HIFAL tries to find a "index.html" file in this folder. If none is found, then its subfolders and files are listed.
  • HIFAL is not multithreaded, which means it can't serve multiple users connecting at the website at once. Instead, it serves only a single user at a time. If this user is downloading a large file, for example, then the whole website is offline to other users until the download is completed.
